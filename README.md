## Running Locally

1. Clone: `git clone [your-repo]`
2. Install: `npm install`
3. Add `.env.local`:
```
   GITHUB_KEY=your_github_token
   GITHUB_AI_KEY=your_github_ai_key
```
4. Run: `npm run dev`
